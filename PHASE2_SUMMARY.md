## Key Insights

### Handle Table Design
```
Simple array-based approach:
- Index = handle ID (generated by counter)
- Value = stored data (i64 or pointer)
- Time: O(1) for both collapse and resolve
- Space: O(n) for n active handles
```

### Bijection Property (Axiom A2)
```
For all values V:
  resolve(collapse(V)) == V

This guarantees reversible computation!
```
