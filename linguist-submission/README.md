# GitHub Linguist Submission Materials for HLX

This directory contains everything needed to submit HLX to GitHub Linguist for official language recognition.

## Files

- **languages.yml** - Entry to add to Linguist's language definitions
- **hlx.tmLanguage.json** - TextMate grammar for syntax highlighting
- **sample.hlxa** - Sample code demonstrating HLX syntax
- **HOW_TO_SUBMIT.md** - Step-by-step submission instructions
- **SUBMISSION_NOTES.md** - Background information about HLX

## Quick Start

1. Fork https://github.com/github/linguist
2. Follow steps in `HOW_TO_SUBMIT.md`
3. Create PR to Linguist repository

## What This Achieves

Once merged, GitHub will:
- Recognize `.hlxa` and `.hlxc` files as HLX code
- Display syntax highlighting in web UI
- Include HLX in repository language statistics
- Make HLX discoverable via GitHub search/explore
