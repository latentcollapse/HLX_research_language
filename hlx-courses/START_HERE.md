# 👋 Welcome to Learn to Code with HLX!

## You're in the Right Place

This course will teach you how to code from absolute zero. No prior experience needed!

## Quick Start (5 minutes)

### Step 1: Make sure HLX is installed

```bash
cd /path/to/hlx-compiler/hlx
cargo build --release
```

Wait for it to build (takes a few minutes the first time).

### Step 2: Test that it works

```bash
./target/release/hlx run /home/matt/hlx-courses/examples/01-hello-world.hlx
```

You should see:
```
"Hello, World!"
0
```

✅ **If you see this, you're all set!**

❌ **If you get an error, check that HLX is installed correctly.**

### Step 3: Start Learning!

Open `README.md` to see the course overview, or jump straight to:

**→ `chapter-01-your-first-program.md`**

## Learning Tips

1. **Go slow** - Understanding is better than speed
2. **Type everything** - Don't copy/paste
3. **Do all exercises** - That's where learning happens
4. **Experiment** - Break things and see what happens
5. **Take breaks** - Let your brain process

## Need Help?

**Stuck on a concept?**
- Re-read the chapter slowly
- Try the example programs
- Check the solutions to exercises
- Experiment with the code

**Getting errors?**
- Read the error message carefully
- Check for typos (computers are picky!)
- Look at the "Common Mistakes" sections
- Compare your code to the examples

**Not sure what to do?**
- Follow the chapters in order
- Don't skip exercises
- Take breaks when frustrated
- Come back with fresh eyes

## What Makes HLX Special for Learning?

1. **Clean syntax** - Easy to read and understand
2. **No surprises** - Programs are deterministic (same input = same output, always)
3. **Instant feedback** - The LSP catches errors as you type
4. **Modern design** - None of the historical baggage of older languages
5. **Real power** - HLX can compile to native code, run on GPUs, work bare-metal
