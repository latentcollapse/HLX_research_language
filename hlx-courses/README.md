# Learn to Code with HLX

**Welcome!** This course will teach you how to code from absolute zero. No prior experience needed.

## What You'll Learn

By the end of this course, you'll be able to:
- Write your own programs that solve real problems
- Understand how computers think and work
- Build games, calculators, and useful tools
- Think like a programmer

## Why HLX?

HLX is **the best language for learning** because:
- **Clean syntax** - Easy to read, easy to write
- **Immediate feedback** - Errors are caught as you type
- **No surprises** - Programs do exactly what you tell them, every time
- **Modern** - Designed in 2025 with everything we've learned about programming

## How to Use This Course

### 1. Set up HLX
```bash
# If you haven't already, build HLX:
cd /path/to/hlx-compiler/hlx
cargo build --release
```

### 2. Start with Chapter 1
Each chapter has:
- **Explanations** - We'll explain every concept clearly
- **Examples** - Real code you can run
- **Exercises** - Practice problems
- **Solutions** - Check your work

### 3. Type Everything Yourself
**Don't copy/paste!** Typing code yourself helps you learn. Make mistakes - that's how you learn!

### 4. Experiment
Change the examples. Break things. See what happens. That's the best way to learn.

## Running Programs

All examples can be run with:
```bash
# From the hlx directory
cd /path/to/hlx-compiler/hlx

# Run any example:
./target/release/hlx run /path/to/example.hlxa
```

## Getting Help

**Stuck?** That's normal! Here's what to do:
1. Re-read the chapter slowly
2. Look at the example code
3. Check the solution
4. Try modifying the solution to understand it better

## Before You Start

You need:
- A computer (Mac, Linux, or Windows)
- HLX installed (see above)
- A text editor (VS Code is recommended)
- Curiosity and patience!
